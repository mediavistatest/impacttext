QR Code Generator

Use this QR Code Generator to create custom QR codes to share with customers and prospects. To create a QR code that will initiate a text message, use this format: SMSTO:+18665578919:Hello, how are you?

When scanned this QR codes will send a text message to 8665578919 with the message, “Hello how are you?”

When you are done creating your code, be sure to test it and then right click to save it so you can share it with contacts.
