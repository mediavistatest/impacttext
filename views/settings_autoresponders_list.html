Autoresponder

Autoresponders are automated text messages that are sent back in reply to incoming texts based on a rule you create. Each autoresponder is activated by a “trigger term” that initiates the process. For example, you could send a link to a coupon back to every incoming text that mentions the term SALE. You can set up autoresponders for each short code or long code that is associated with your account.


You can set up to 5 custom autoresponder rules. Autoresponder messages can be sent on a timed schedule by using the Delay Rule option. For example, if you would like to send out an automatic message to anyone who texts the word SALE immediately and then again 2 days after that, set your Delay Rule to 48 hours.
